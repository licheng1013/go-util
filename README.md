# go-util
## 介绍
- 2022/8/8
- 一个简单的go-util工具库,简易封装一些api来使用。
- 目前命名规则并不完善，欢迎加入并且共享。

## 版本变化
- 小版本迭代
- v0.0.1 > v0.0.2 废弃和优化
- 中版本迭代
- v0.0.0 > v0.1.0 增加新工具api
- 大版本迭代
- v0.0.0 > v1.0.0 删除废弃的api 


## 安装教程
- Github安装
- 无

- Gitee安装
- 以下是配置格式
```text
require (
	go-util v0.0.4
)

replace go-util => gitee.com/licheng1013/go-util v0.0.4
```
